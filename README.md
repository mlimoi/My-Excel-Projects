# Excel Data Analysis Projects

## Overview

This repository contains Excel projects completed during my **Level 3 Data Technician Skills Bootcamp**, alongside an independent Football Manager wage analysis project.

The projects demonstrate my ability to clean, organise, analyse and visualise data using formulas, PivotTables, charts, slicers and statistical functions.

## Skills Demonstrated

* Data cleaning and organisation
* Excel formulas and functions
* Filtering and sorting
* PivotTables and PivotCharts
* Interactive slicers
* Conditional formatting
* Data categorisation
* Statistical analysis
* Dashboard and chart creation
* Communicating findings through data visualisation

### 4. Product Sales Categorisation

Organised product sales into performance categories using formulas and PivotTables.

**Tasks completed:**

* Organised data by county and product
* Created a PivotTable showing sales volumes
* Used the `SWITCH` function to categorise results
* Grouped sales as High, Medium or Low
* Applied formulas across the full dataset

```excel
=SWITCH(TRUE,C2>600,"High",C2>=300,"Medium","Low")
```
